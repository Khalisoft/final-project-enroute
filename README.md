## Description

Site-Rider ( formally, enroute) is a carpooling app to optimise transportation of people, parcels and vehicles across remote mine site operations.

## Technology Stack
* HTML / CSS / Javascript

* Mongoose - NoSQL database
* MongoDB

* Node JS

* Pusher.js - real-time notifications
